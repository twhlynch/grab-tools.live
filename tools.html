Tools

This is a collection of simple tools to use out of the JSON Level Editor. The level compiler will merge levels, grouping each part so they arent mixed up. The bulk signs tool will generate a long line of signs from text input. The pixel art tool does what it says it does.

Advertisement

Support the site by disabling your adblocker. The ads are non-intrusive and help keep the site running.

Level Compiler

Select multiple level files and combine them into one file. Each level will be grouped.

Pixel Art

Specify the quality, and convert an image to a grab level.

Text To Signs

Convert text into a line of signs horizontally or vertically.

Text To Signs Animated

Convert text into animated signs.

Multi Download

Paste a list of level browser links to download all the maps.
DO NOT STEAL MAPS!

Custom Colors

Drag this to your bookmarks bar or click it to copy it and paste it into a bookmark. Then go to the level browser, log in, and click it to run.

Custom Color Picker

Level to JSON

Convert a level to JSON format or from JSON back to protobuf.

Outline effect

Obj to point cloud

Generate a cloud of vertices from an obj 3d model

Level Exploder

Move objects in a map away from each other
